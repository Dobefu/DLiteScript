Sorts a number array in ascending order.

## Examples

```go
var arr []number = [3, 1, 4, 1, 5, 9, 2, 6]
arr = arrays.sortNumbers(arr)
printf("%s\n", arr) // [1, 1, 2, 3, 4, 5, 6, 9]
```
