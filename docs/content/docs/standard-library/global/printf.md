Prints a formatted string to standard output.

## Examples

```go
printf("Hello, %s!\n", "world") // Hello, world!
printf("Number: %g\n", 42)      // Number: 42
printf("Boolean: %t\n", true)   // Boolean: true
```

## Format Specifiers

- `%s` - String
- `%g` - Number
- `%t` - Boolean
